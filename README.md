# 🛡️ Blue Shield Elite
Navegador de ciberseguridad con motor Chromium personalizado.

### 🚀 Funciones:
- **Multitarea:** Pestañas reales independientes.
- **Login:** Base de datos de usuario local.
- **Auditoría:** Escáner de IP privada y pública integrado.
- **Bypass:** Todo se abre dentro de la App.

### 🛠️ Instalación:
1. `pip install PyQt6 PyQt6-WebEngine requests`
2. `python main.py`

**Blue Shield** no es solo un navegador; es un centro de mando integral diseñado para la productividad, el desarrollo y la ciberseguridad. Creado para centralizar herramientas de IA, entornos de programación y utilidades de red en una interfaz única, optimizada y segura.

---

## 🚀 Características Principales

* **⚡ Hub de Inteligencia Artificial:** Acceso directo y optimizado a **Grok AI**, **AI Studio** y herramientas de generación como **Lovable.dev**.
* **🛡️ Sistema de Bypass Inteligente:** Lógica avanzada de detección de cabeceras `X-Frame-Options`. Si una web prohíbe el enmarcado (como Google o GitHub), Blue Shield la lanza en una *Sandboxed Window* automáticamente para evitar bloqueos de conexión.
* **📂 Dev & Hack Suite:** Integración nativa con **JSFiddle**, **IP Logger**, **Technical City** y escaneos de seguridad.
* **🎨 Interfaz Elite:** Sidebar inteligente con expansión por *hover*, diseño *Glassmorphism* y modo oscuro profundo para reducir la fatiga visual durante sesiones largas de cracking o coding.
* **🔍 Home Search:** Pantalla de inicio minimalista con motor de búsqueda híbrido integrado.

---

## 🛠️ Stack Tecnológico

* **Frontend:** HTML5, CSS3 (Custom Variables, Flexbox, Animations) y JavaScript (ES6+).
* **Backend (Versión Desktop):** Python 3.x + PyQt6 + QtWebEngine (Motor Chromium).
* **Security:** Lógica de filtrado de URLs y protección de navegación.
